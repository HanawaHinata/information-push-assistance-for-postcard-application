# "生日贺卡信息推送辅助程序"的下载器。

## 概述
本程序只是个人制作用来取乐的，只有从站点下载压缩包解压、安装应用程序的功能。

在程序开头添加了身份认证功能，仅限于本地验证和是否是36。

压缩包存储的服务器使用了呆萌网盘的阿里云服务器。

## 开发环境
本应用使用AIDE开发。环境兼容IntelliJ的IDEA和Android Studio。
